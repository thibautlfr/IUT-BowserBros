\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_sound_manager}{Sound\+Manager}} class manages the playback of various sounds and music in the application. 

The \mbox{\hyperlink{class_sound_manager}{Sound\+Manager}} class provides functionality to play jump sound effects, main music, menu music, death music, win music, and level passed music. It also allows setting the main and effects volume, stopping all sounds, and retrieving volume levels.
